Related Posts

A liquid loop to go through the category that also matches the page 'subcat' to only show posts related to current theme or category:

    {% if page.category == 'support_themes' %} {% for post in site.categories.support_themes %} {% if post.subcat == page.subcat %}
  • {{ post.title }}
    • {% endif %} {% endfor %} {% else %} {% for post in site.posts %} {% if post.category == page.category %}
  • {{ post.title }}
    • {% endif %} {% endfor %} {% endif %}
